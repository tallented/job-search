## Search Scope

## Output Wanted

- `Apply Now`
- `Apply If Time`
- `Skip / Probably Not`

Each item should include:

- company
- role title
- work model
- compensation if available
- one-line fit assessment
- key stretch or risk
- source link

## Good Decision Rule

Only interrupt the current queue if a new role is:

- clearly stronger than `A Place for Mom` or `Vanta`
- in the preferred leadership lane
- compensation-compatible or plausibly negotiable
- posted recently enough that applying early is valuable
