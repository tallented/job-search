# Queue Triage - 2026-04-23

The backlog is now large enough that the default should be:

- fully tailor only the top `4-5` roles
- submit the next tier with `no more resume work`
- archive the bottom tier instead of preserving optionality forever

The key change is process discipline. A role should now earn customization. Everything else should either go out fast or get dropped.

## Recommended Working Rule

- `Full tailor`: only for roles that are strong on compensation, title coherence, fit, and seriousness
- `Direct submit`: no more bullet surgery, no more line-wrap polishing, no more summary rewriting
- `Archive`: anything down-level, below floor, off-lane, or stale enough that it is not worth another cycle

Important nuance:

- For roles already sitting in `CREATED`, the fastest path is usually to submit the existing tailored packet as-is.
- Swapping back to a pure canonical resume only makes sense if the current tailored file is weak, unfinished, or sending the wrong signal.

1. Keep at most `4-5` roles in `full tailor`.
